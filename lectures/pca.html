¿Por qué necesitamos PCA?

El problema de la dimensionalidad

Con muchas variables, el número de relaciones crece explosivamente:

  • Con 20 variables → \(\binom{20}{2} =\) 190 correlaciones posibles
  • Con 100 variables → 4,950 correlaciones posibles
  • Revisar cada una es impráctico

¿Qué es PCA?

¿Qué es PCA?

PCA transforma variables correlacionadas en un conjunto más pequeño de variables no correlacionadas llamadas componentes principales, conservando la mayor información posible.

Ventajas de PCA

Ventaja Descripción
🔻 Reducción de dimensionalidad De \(p\) variables a \(k \ll p\) componentes
🔗 No correlación Los componentes son ortogonales entre sí
📊 Resumen de información Las primeras componentes capturan la mayor varianza
🔇 Eliminación de ruido Las últimas componentes suelen ser ruido

Note

Regla práctica: en la mayoría de datasets reales, las primeras 2–5 componentes explican el 80–95 % de la varianza total.

Ejemplo: perfil de salud de pacientes

Eigenvectores y Eigenvalores

Repaso de Álgebra Lineal

Dada una matriz cuadrada \(A\), un eigenvector \(\mathbf{v}\) satisface:

\[A\mathbf{v} = \lambda \mathbf{v}\]

  • \(\mathbf{v}\): dirección que no rota bajo la transformación de \(A\)
  • \(\lambda\): escalar que indica cuánto se escala esa dirección

Ejemplo numérico

Sea \(A = \begin{pmatrix} 2 & 0 \\ 0 & 3 \end{pmatrix}\)

Vector \(\mathbf{v}_1 = (1, 0)^T\):

\[A\mathbf{v}_1 = \begin{pmatrix}2\\0\end{pmatrix} = 2\begin{pmatrix}1\\0\end{pmatrix}\]

✅ Eigenvector con \(\lambda_1 = 2\)

Vector \(\mathbf{v}_2 = (0, 1)^T\):

\[A\mathbf{v}_2 = \begin{pmatrix}0\\3\end{pmatrix} = 3\begin{pmatrix}0\\1\end{pmatrix}\]

✅ Eigenvector con \(\lambda_2 = 3\)

Important

¿Es \((1,1)^T\) eigenvector de \(A\)?
\(A(1,1)^T = (2,3)^T \neq \lambda(1,1)^T\) para ningún \(\lambda\). ❌

Pasos del PCA

Los 5 pasos del PCA

Paso 1: Estandarización

\[Z_i = \frac{X_i - \bar{X}}{s}\]

¿Por qué? Variables con mayor escala dominarían el PCA aunque no sean más importantes.

import numpy as np

X     = np.array([2.0, 4.0, 6.0])
media = np.mean(X)
desv  = np.std(X, ddof=1)
Z     = (X - media) / desv

print(f"X:          {X}")
print(f"Media:      {media:.2f}   Desv. estándar: {desv:.4f}")
print(f"Z:          {np.round(Z, 4)}")
print(f"Media de Z: {np.mean(Z):.10f}  (≈ 0)")
print(f"Desv de Z:  {np.std(Z, ddof=1):.4f}    (= 1)")

Paso 2: Matriz de covarianza

\[\text{cov}(x_j, x_k) = \frac{1}{n-1}\sum_{i=1}^{n}(x_{ij}-\bar{x}_j)(x_{ik}-\bar{x}_k)\]

Para \(p\) variables, \(\Sigma\) tiene dimensión \(p \times p\). La diagonal son varianzas; fuera, covarianzas.

import numpy as np

x1 = np.array([1, 2, 3])
x2 = np.array([2, 4, 6])
x3 = np.array([1, 3, 5])

X     = np.column_stack([x1, x2, x3])
Sigma = np.cov(X.T)

print("Matriz de covarianza Σ:")
print(Sigma)
Matriz de covarianza Σ:
[[1. 2. 2.]
 [2. 4. 4.]
 [2. 4. 4.]]

Note

¿Qué implica que \(\text{cov}(x_1, x_2) = 2\)? ¿Y que \(x_2\) y \(x_3\) tengan la misma covarianza con \(x_1\)?

Paso 3: Eigenvalores

Se resuelve la ecuación característica:

\[\det(A - \lambda I) = 0\]

Ejemplo: \(A = \begin{pmatrix} 5 & 2 \\ 2 & 5 \end{pmatrix}\)

\[\det\begin{pmatrix} 5-\lambda & 2 \\ 2 & 5-\lambda \end{pmatrix} = (5-\lambda)^2 - 4 = 0\]

\[\lambda^2 - 10\lambda + 21 = (\lambda-7)(\lambda-3) = 0\]

\[\boxed{\lambda_1 = 7 \qquad \lambda_2 = 3}\]

Paso 3: Eigenvectores

Para cada \(\lambda_i\) se resuelve \((A - \lambda_i I)\mathbf{v} = \mathbf{0}\)

Para \(\lambda_1 = 7\):

\[\begin{pmatrix}-2&2\\2&-2\end{pmatrix}\mathbf{v}=\mathbf{0} \;\Rightarrow\; y=x\]

\[\mathbf{u}_1 = \frac{1}{\sqrt{2}}\begin{pmatrix}1\\1\end{pmatrix}\]

Para \(\lambda_2 = 3\):

\[\begin{pmatrix}2&2\\2&2\end{pmatrix}\mathbf{v}=\mathbf{0} \;\Rightarrow\; y=-x\]

\[\mathbf{u}_2 = \frac{1}{\sqrt{2}}\begin{pmatrix}1\\-1\end{pmatrix}\]

Tip

En PCA: \(\mathbf{u}_1\) (mayor \(\lambda\)) = primera componente principal — dirección de mayor varianza.

Geometría del PCA

Paso 4: Variabilidad explicada

\[\text{Variabilidad}_i = \frac{\lambda_i}{\displaystyle\sum_{j=1}^{p} \lambda_j} \times 100\%\]

Ejemplo: \(\lambda_1 = 7,\; \lambda_2 = 3 \;\Rightarrow\; \lambda_{\text{total}} = 10\)

Componente Eigenvalue % Varianza % Acumulado
PC1 7 70 % 70 %
PC2 3 30 % 100 % 
lambdas     = np.array([7, 3])
porcentajes = lambdas / lambdas.sum() * 100
acumulado   = np.cumsum(porcentajes)

for i, (l, p, a) in enumerate(zip(lambdas, porcentajes, acumulado), 1):
    print(f"PC{i}: λ={l}  →  {p:.1f}%  (acum: {a:.1f}%)")
PC1: λ=7  →  70.0%  (acum: 70.0%)
PC2: λ=3  →  30.0%  (acum: 100.0%)

PCA con sklearn

PCA con sklearn en Python

from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA
import numpy as np

X = np.array([
    [2,  5, 48],
    [3,  7, 55],
    [4,  8, 60],
    [5, 10, 65],
    [6, 11, 72],
], dtype=float)

X_std  = StandardScaler().fit_transform(X)
pca    = PCA()
scores = pca.fit_transform(X_std)

print("Eigenvalores:         ", np.round(pca.explained_variance_, 4))
print("% Varianza:           ", np.round(pca.explained_variance_ratio_ * 100, 2))
print("Eigenvectores (filas):")
print(np.round(pca.components_, 4))

Caso real: Dataset Iris

from sklearn.datasets import load_iris

iris        = load_iris()
X_std_iris  = StandardScaler().fit_transform(iris.data)
pca_iris    = PCA(n_components=2)
scores_iris = pca_iris.fit_transform(X_std_iris)

print("Varianza explicada:")
for i, p in enumerate(pca_iris.explained_variance_ratio_ * 100, 1):

print(f"  Total: {sum(pca_iris.explained_variance_ratio_)*100:.2f}%")

Limitaciones de PCA

Limitación Descripción
Linealidad Solo captura relaciones lineales
Interpretabilidad Combinaciones de todas las variables
Outliers PCA es sensible a valores atípicos
Escala Requiere estandarización previa
Correlación Si no hay correlación, PCA no ayuda

Alternativas no lineales: t-SNE, UMAP, Kernel PCA.
