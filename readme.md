# Notas del curso: Almacenes y Minería de Datos (2026-2) — Facultad de Ciencias, UNAM

Este repositorio contiene las **notas, ejemplos reproducibles y materiales de apoyo** del curso **Almacenes y Minería de Datos** (semestre **2026-2**) impartido en la **Facultad de Ciencias, UNAM**.  
El objetivo es ofrecer un recurso **académico y técnico**, con énfasis en **modelado dimensional**, **procesos ETL**, **calidad de datos**, y **técnicas fundamentales de minería de datos**, acompañado de implementaciones en Python y materiales en Quarto.

---

## Objetivos del curso

Al finalizar el curso, el estudiante será capaz de:

- Comprender el rol de un **Data Warehouse (DW)** y su diferencia con sistemas operacionales (OLTP).
- Diseñar **esquemas dimensionales** (Kimball) y discutir arquitecturas alternativas (Inmon y enfoques híbridos).
- Implementar un flujo **ETL/ELT** reproducible con validación de calidad y homologación de fuentes.
- Construir **Data Marts** integrados mediante **dimensiones conformadas** y **bus matrix**.
- Realizar análisis exploratorio y aplicar técnicas de **minería de datos**: clasificación, clustering, reducción de dimensionalidad, reglas de asociación y evaluación.
- Comunicar resultados con rigor: métricas, validación, interpretación y visualización.
